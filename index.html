What is GitHub Pages?

GitHub Pages allows you to host static websites directly from a GitHub repository. It's a great way to showcase projects, create personal portfolios, or share documentation.

How to Access the Page

Once the repository is set up with GitHub Pages, you can view the site at:

https://<your-github-username>.github.io/<repository-name>/

For example, if your username is example-user and the repository is test-repo, the site will be live at:

https://example-user.github.io/test-repo/
